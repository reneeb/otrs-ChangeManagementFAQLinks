=head1 NAME

ChangeManagementFAQLinks - enable linking ITSMChangeManagement objects with FAQ entries

=head1 DESCRIPTION

This package just adds a few SysConfig options to enable linking of FAQ entries with
ITSM Changes and ITSM WorkOrders.
